== Description ==

Used to overlay images on the current page.

You can check for more information and samples <a href="http://www.pedrolamas.com/projectos/jquery-lightbox-en/">here</a>.

== Installation ==

1. Just download the file and unzip.
1. Upload `jquery-lightbox-balupton-edition` directory to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress

== Faq ==

Every time you want to use the Lightbox effect, just place a link to the image, and add the <i>rel=&quot;lightbox&quot;</i> attribute to it, in order to activate the effect when the link is clicked.
You can also use <i>rel=&quot;lightbox-album&quot;</i> replacing <i>album</i> by a any keyword to group some pictures on an album!

You can check for more information and samples <a href="http://www.pedrolamas.com/projectos/jquery-lightbox-en/">here</a>.
